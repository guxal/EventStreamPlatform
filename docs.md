# Documentation Index

This repository uses the following documentation structure:

- `README.md`  
  High-level repository overview and current implementation status.

- `stakeholder_program_overview.md`  
  Executive/high-level program view for stakeholders (scope, status, risks, roadmap).

- `docs_marketing_copilot.md`  
  Deep technical architecture and current implementation flow for engineers.

- `AGENTS.md`  
  Canonical implementation guide and operational instructions for AI/code agents.

## Recommended reading order
1. Stakeholders / Product: `stakeholder_program_overview.md`
2. Engineers: `README.md` then `docs_marketing_copilot.md`
3. AI/code contributors: `AGENTS.md`
